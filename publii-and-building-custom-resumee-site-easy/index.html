    Publii and building custom resumee site? Easy :)

    Mariusz Suchodolski June 26, 2019
    Publii is one of those great static site generators that I personally prefer over Wordpress, Joomla and all those heavyweight CMS systems. Why? It lets you build a simple static website that saves you the hassle of updating it every-so-often. Compared to Wordpress, you have no ingress points for the attacker. SQL injection? There is no SQL DB. Stealing cookies? No cookies to be stolen anyways. The fact that it's basically all HTML makes it so great. You can compress everything, easily make it AMP friendly without the hassle of installing tons of plugins.
